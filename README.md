# Portfolio de Mouaad Belemhouar

Portfolio personnel bilingue FR/EN, con&ccedil;u pour un partage simple avec les recruteurs et les ressources humaines.

## Mise en ligne avec GitHub Pages

1. Cr&eacute;ez un nouveau d&eacute;p&ocirc;t sur GitHub.
2. Ajoutez tous les fichiers de ce dossier dans le d&eacute;p&ocirc;t.
3. Envoyez-les sur la branche `main`.
4. Ouvrez l'onglet `Actions` du d&eacute;p&ocirc;t et laissez le workflow `Deploy Portfolio to GitHub Pages` s'ex&eacute;cuter.
5. Une fois le d&eacute;ploiement termin&eacute;, votre site sera disponible sur un lien GitHub Pages.

## Structure

- `index.html` : page principale du portfolio
- `styles/main.css` : design principal et responsive
- `scripts/main.js` : bilingue FR/EN et interactions
- `assets/images/` : photo de profil
- `assets/docs/` : CV et documents
- `.github/workflows/deploy-pages.yml` : d&eacute;ploiement automatique sur GitHub Pages
